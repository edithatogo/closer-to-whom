# ADR 0004: Reference oracle, JAX support, Mojo promotion gate

- Status: accepted
- Date: 2026-07-11

NumPy/Polars is the reference oracle. JAX is an optional supported accelerator after differential checks. Mojo is developed as a first-class experimental backend but remains non-critical until pinned-toolchain, numerical, reproducibility, benchmark, packaging, and maintenance gates pass.
