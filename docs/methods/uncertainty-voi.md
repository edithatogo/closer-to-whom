# Uncertainty and value of information

## Deterministic sensitivity analysis

One- and two-way analyses vary treatment frequency and duration, facility evidence threshold, within-area origins, route engine/time, transport mode, vehicle and parking costs, accompaniment, time valuation, uptake, residual hospital visits, provider travel, capacity envelopes, and adoption speed.

## Probabilistic sensitivity analysis

The default engine uses scrambled Sobol draws with declared marginal and joint distributions. Every run records seed, draw count, parameter fingerprint, backend, and numerical precision. Report simulation intervals, probability optimal, expected regret, equity-condition probabilities, and site-selection probabilities.

## Structural uncertainty

Facility-network evidence sets, geographic representation, mode availability, clinical pathway structure, capacity regime, funding/commissioning state, and adoption model are reported as separate structural scenarios rather than pooled into one interval.

## Value of information

Primary consequences remain disaggregated. VOI requires a declared scalar decision value, so several perspective-specific value functions are used. Report EVPI, EVPPI, EVSI, ENBS, and break-even research cost.

The microdata question is explicit: a new individual-level study is supported only when it has positive expected net benefit, materially may change the preferred configuration or equity conclusion, and cannot be replaced by cheaper public or aggregate evidence. Model structural uncertainty and ethical/governance burden are included in the research decision narrative.
