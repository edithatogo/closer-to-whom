# Ethics and policy-evaluation scope

The intended first phase is an independent ex-ante service-policy evaluation and mathematical model using only publicly accessible aggregate information. It recruits no participants, contacts no patients or whānau, intervenes in no care, accesses no confidential system, and performs no individual linkage.

The project should still complete the applicable New Zealand ethics-scope screening and retain the resulting determination. An out-of-scope determination is not represented as ethical approval. Local institutional, publication, cultural, and data-governance expectations remain relevant.

A later microdata phase would be a separate protocol with separate data governance, privacy, Māori data-governance consideration, ethics/scope review, security controls, and analysis plan. The VOI component determines whether that escalation is likely to be worth its cost and burden.
